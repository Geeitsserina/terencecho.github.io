Arduino Quadcopter

Currently working on

I am currently building a functional quadcopter based off an Arduino microcontroller and recieves command through a specialized Android app that I developed (blueQuad) which utilizes Bluetooth technology. As of now, the quadcopter is using a makeshift frame that I plan on replacing with a frame that I have designed using SolidWorks. I am currently in the process of solving for the best combinations of parameters for my PID controller in order to keep the quadcopter stabilized in the air. The code for the Arduino microcontroller can be viewed on my Github.
